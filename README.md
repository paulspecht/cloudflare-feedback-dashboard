# Cloudflare Feedback Dashboard (Prototype)

This project is a prototype feedback analysis dashboard built using Cloudflare Workers.

## Stack
- Cloudflare Workers (API + UI)
- Cloudflare Workers AI (sentiment + bot risk classification)
- Cloudflare D1 (feedback storage and analytics)

## Live Demo
https://my-first-worker.paulspecht.workers.dev

## Features
- Feedback ingestion API
- AI-based sentiment classification
- Aggregated analytics (by product, source, sentiment)
- Lightweight dashboard UI

This project was built as part of the Cloudflare Product Manager Intern assignment.
